# ACTIVIDAD-3-FLUJO-DE-CONTROL-PART.-2

Tarea: Utilizando bucles, crea un programa que permita al usuario introducir las 4 calificaciones
de n cantidad de estudiantes, calcular el promedio y determinar si aprobó o reprobó, el
resultado en pantalla debería ser de esta manera.

En el pseudo codigo que realice usamos un blucle para que nos pida las 4 calificaciones de x numeros de estudiantes y al final se sumen y se dividan entre 4 para saber si aprobaron o reprobaron, podran aprobar si su calificacion es >=70 y reprueban si es <70
